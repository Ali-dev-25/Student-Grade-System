# Student Grades Application

## Description
Desktop application built with Python and Flet to register and display ICT students grades
using SQLite database.

## Technologies
- Python
- Flet
- SQLite3

## Features
- Add student grades
- Store data in database
- Calculate average automatically
- Display student performance

## How to Run
pip install flet  
python main.py
